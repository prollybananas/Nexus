# ইথেরিয়াম ব্যবহার করা {#using-nexus}

<div class="featured">

**ইথেরিয়াম ব্যবহার শুরু করতে চান?** ইথেরিয়ামে তৈরি করা কয়েক শত বিকেন্দ্রীভূত অ্যাপ্লিকেশন (ড্যাপ) আছে যেগুলো আপনি আজই ব্যবহার করতে পারেন।
